HTML and CSS Quick-Reference Guide


HTML tags

  1. <html> : Defines the start and end of an HTML document. It's the root element of the page. 
    • <html>

  <head>
    <title>Sample Page</title>
  </head>
  
  <body>
    <h1>Hello, World!</h1>
  </body>

</html>
  2. <head> : Contains metadata (information about the document like title, character set, linked stylesheets, etc.) 
    • <head>
  <meta charset="UTF-8">
  <title>My Page</title>
</head>
  3. <title> : Defines the title of the webpage, shown in the browser's title bar or tab. 
    • <head>
  <title>My Web Page</title>
</head>
  4. <body> : Contains the content of the webpage such as text, images, links, and other elements. 
    • <body>
  <h1>Welcome to My Website</h1>
  <p>This is a paragraph of text.</p>
</body>
  5. <h1> to <h6> : Defines header tags, with <h1> being the highest (largest) and <h6> the lowest (smallest). 
    • <h1>Main Header</h1>
<h2>Sub Header</h2>
<h3>Smaller Header</h3>
  6. <p> : Defines a paragraph of text. 
    • <p>This is a paragraph of text.</p>
  7. <a> : Defines a hyperlink to link to other pages or resources. It's used with the href attribute. 
    • <a href="https://www.example.com">Click here to visit Example</a>
  8. <img> : Defines an image on the page. It uses attributes like src (source) and alt (alternative text). 
    • <img src="image.jpg" alt="A description of the image">
  9. <ul> and <ol> : Defines an unordered (bulleted) or ordered (numbered) list. Items inside are wrapped with <li> tags. 
    • <ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

<ol>
  <li>First Item</li>
  <li>Second Item</li>
  <li>Third Item</li>
</ol>
  10. <li> : Defines a list item in an unordered or ordered list. 
    • <ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

CSS Properties

  1. color : Sets the text color of an element. 
    • p {
  color: blue;
}
  2. background-color : Sets the background color of an element. 
    • div {
  background-color: lightgray;
}
  3. font-size : Defines the size of the text. 
    • h1 {
  font-size: 24px;
}
  4. font-family : Specifies the font type of the text. 
    • body {
  font-family: Arial, sans-serif;
}
  5. margin : Defines the space outside an element. 
    • div {
  margin: 20px;
}
  6. padding : Defines the space inside an element, between content and border. 
    • div {
  padding: 15px;
}
  7. border : Specifies the border of an element. 
    • div {
  border: 2px solid black;
}
  8. width & height : Sets the width and height of an element. 
    • img {
  width: 200px;
  height: 150px;
}
  9. display : Specifies how an element is displayed on the page. 
    • div {
  display: flex;
}
  10. position : Defines how an element is positioned in the document. 
    • div {
  position: absolute;
  top: 50px;
  left: 100px;
}
