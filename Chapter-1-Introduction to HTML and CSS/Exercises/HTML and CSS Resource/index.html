HTML tags

  1. The <html> tag : Used to wrap the entire html document.
    • <html>

    <body> </body>

    <body> </body>

</html>
  2. The <head> tag : Used to provide metadata(meta-information) about the document such as its title, character encoding, linked stylesheets, etc.
  3. The <h> tag : Used to display headings where <h1> is the biggest and <h6> the smallest.
    • <h1> I am the biggest heading </h1>
    • <h6> I am the smallest heading </h6>
